Features
Everything you need,
nothing you don't

SMS + RCS

Send and receive text messages directly from your desktop. Supports SMS, RCS, and group chats through the Google Messages protocol.

Runs locally

Your messages never leave your machine. SQLite database, local binary, no cloud account required. You own your data completely.

MCP tools

Seven structured tools for AI assistants via Model Context Protocol. Search messages, send texts, list conversations - all from Claude Code.

Full-text search

Search across all your messages instantly. Find any conversation, any text, from any contact. Results update in real time.

Native macOS app

A real .app with menu bar presence, native notifications, and auto-launch. Not Electron — a Swift app with WebKit. 28 MB universal binary.

Open source

MIT licensed. Swift + Go, no dependencies to install. Built on the battle-tested mautrix/gmessages library.

Quick start
Up and running in 60 seconds
1

Download and install

Download OpenMessages.dmg, open it, and drag to Applications. On first launch, right-click → Open to bypass Gatekeeper (it's unsigned). Universal binary — Apple Silicon and Intel.

2

Pair your phone

Launch the app. It shows a QR code. On your Android phone, open Google Messages → Settings → Device pairing and scan it. One-time setup — your session persists across restarts.

3

Start messaging

Your conversations sync instantly. Send and receive SMS/RCS from the desktop. The app lives in your menu bar with native notifications.

4

Add AI integration

Add to ~/.mcp.json for Claude Code:

{
  "mcpServers": {
    "openmessages": {
      "command": "/Applications/OpenMessages.app/Contents/Resources/openmessages",
      "args": ["serve"]
    }
  }
}
